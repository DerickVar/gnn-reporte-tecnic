# Directorio de Referencias - Redes Neuronales de Grafos (GNN)

Este directorio contiene las referencias bibliográficas organizadas por temas relacionados con Redes Neuronales de Grafos.

## Estructura de Carpetas

- **gnn_fundamentos/** - Trabajos fundamentales sobre GNN
- **gnn_dinamicos/** - Redes de grafos dinámicos y temporales
- **aplicaciones/** - Aplicaciones prácticas (tráfico, energía, recomendación)
- **tecnicas_avanzadas/** - Técnicas avanzadas (transformers, meta-learning)
- **deteccion_comunidades/** - Detección de comunidades y estructura
- **prediccion_enlaces/** - Predicción de enlaces y embeddings
- **otros/** - Otras aplicaciones y trabajos relacionados

## Archivos

- `referencias.bib` - Archivo BibTeX completo con todas las referencias
- `referencias_por_tema.md` - Organización de referencias por tema

## Cómo usar

1. Selecciona el archivo BibTeX correspondiente al tema que necesites
2. Importa las referencias en tu gestor de citas favorito (Mendeley, Zotero, etc.)
3. Cita usando el ID de la referencia en LaTeX
